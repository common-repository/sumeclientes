Plugin que permite integrar las páginas de captura diseñadas en la plataforma Sume Clientes con tu sitio de Wordpress.
 
== Description ==
 
Este plugin permite que tus páginas de captura diseñadas en la plataforma Sume Clientes puedan ser vistas desde tu sitio web,
facilitando la integración y publicación de estas para sacar el mejor provecho a las herramientas de mercadeo en línea que
Sume Clientes te provee y de esta forma puedas lograr tu objetivo de obtener más clientes.
 
== Installation ==
 
1. Subir la carpeta descomprimida 'sumeclientes' al directorio '/wp-content/plugins/'.
2. Activar el plugin desde el menú 'Plugins' de wordpress.
3. Ingresar la API Key generada en la plataforma de Sume Clientes.
